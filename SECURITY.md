# Security Policy

Bluetooth Assistant works with Bluetooth discovery, nearby devices, and runtime permissions. Security and privacy reports are welcome.

## Supported Versions

Security fixes should target the latest code on the `master` branch.

## Reporting A Vulnerability

If you find a security issue, please avoid opening a public issue with exploit details. Contact the maintainer through the GitHub profile or open a minimal issue asking for a private contact channel.

Useful details include:

- Affected Android version and device model.
- Bluetooth device type or module.
- Steps to reproduce.
- Logs or proof-of-concept details.
- Suggested mitigation, if known.

## Areas Of Interest

- Android Bluetooth permission handling.
- Unexpected exposure of device identifiers.
- Unsafe handling of received Bluetooth messages.
- Background service behavior.
- Third-party SDK configuration.
